## Technology Stack

Here's some information about Flickmemo's tech stack. This session is being written, so some things may change.

### Mobile App:

 - Flutter (3.13.2);
 - Dart (3.1.0);
 - Firebase.

### API:

  - Ruby (3.2.2);
  - Rails (7.0.7);
  - Docker and `docker-compose`, for API and DB;
  - PostgreSQL;
  - RSpec, for testing;
  - Heroku.
